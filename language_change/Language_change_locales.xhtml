reading-1511 TTS Change Languages Automatically (main language code

Screen readers and other Assistive Technology that use TTS should be able to detect language changes marked in XHTML content. A screen reader should be able to be configured to automatically change the TTS speaking language when it encounters a language change in the XHTML. In some cases available voices are dependant of the operating system. to perform this test, tester must be sure that the following languages are installed on his computer:

  • English, US
  • French, Canada

Configure your screen reader or other Assistive Technology to automatically change the TTS language. Read the phrases below line by line and continuously. The test passes if the TTS language changes as it encounters the language change.

Indicate Pass or Fail.

Language Change Test

In US English, "Good morning my darling."; In french Canadian, "Bonjour ma chérie."

End of language change testing.
