math-090 MathML with Fallback PNG image in MathML Annotation on a reading system which supports MathML

The tests checks the reading experience of MathML with annotation. Please skip this test if your reading system does not support MathML.

The test passes if all the following are true:

  • The assistive technology reports presence of the MathML expression and provides functionality to read it,
  • The assistive technology does not report presence of fallback image with alttext inside the AnnotationXML

The test fails if you do not see or hear any math equation or fallback image with alttext is reported.

Test 090

Math Equation will appear between the following two lines.

y − y 1 = y 2 − y 1 x 2 − x 1 ( x − x 1 )

Additional Notes for Test 090

If the test passes, please provide notes about your MathML reading experience, and how well you were able to navigate in MathML.

Please make a note if you are able to see both the mathML rendered equation as well as its fallback png image.
