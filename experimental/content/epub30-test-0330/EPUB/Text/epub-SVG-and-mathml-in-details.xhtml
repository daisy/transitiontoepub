SVG alt-text and MathML inside details tests

The test has an SVG with alt text and aria-details linking to a details element with the MathML inside the details. The summary says "MathML Presentation."

math-080 Support for SVG alt-text and MathML inside details

The test passes if the assistive technology reads the Alt text of the SVG image, and after expanding the details and summary, the Assistive Technology reports presence of the MathML expression and provides functionality to read it.

The test fails if The MathML in the details is presented as math markup without rendering it, i.e. it is garbage.
