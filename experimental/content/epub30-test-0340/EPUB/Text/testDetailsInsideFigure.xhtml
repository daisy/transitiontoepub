Extended description using details and summary inside Figure Element

Notes about techniques

For the image on this page, the HTML details and summary elements are used to expose the extended image descriptions directly outside the figcaption but still within the figure element for the image.

We have also included aria-details whcih points to this details/summary structure, which eventually will be picked up by assistive technologies and reading systems to make this a seamless experience.

The test below refer to the bar chart image immediately following.

extdesc-240Detection and utilization of details and summary elements inside figure element but outside the figcaption

This test passes if all the following steps are successful:

  1. The HTML details/summary elements are reported. (For example, there is an interactive element like button, link etc. that you can activate.)
  2. You can expand and collapse the details element containing the extended description. (For example, you can activate the interactive element to expand or collapse the extended description.)
  3. When expanded, the extended description should appear immediately after the activating element. When you collapse it, you should remain at the same position.
  4. The table within the extended description in the details is accessible to assistive technologies and is navigable.
